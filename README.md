## 使用步骤

### 1. 命令行 mkdir demo && cd demo (新建一个demo目录并进入demo目录)

### 2. git clone https://github.com/zhuanshenai/vue_ele_demo.git (克隆项目地址)

### 3. cd vue_ele_demo (进入项目目录)

### 4. npm install / cnpm install(安装了淘宝npm镜像后可以使用cnpm安装项目依赖)

### 5. npm run dev (运行项目)
